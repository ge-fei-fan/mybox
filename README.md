# mybox

使用gin框架实现文件存储网盘


## 功能支持

- [x] 用户模块
    - [x] 密码登录
    - [ ] 刷新Authorization
    - [x] 用户详情
    - [ ] 用户容量
- [x] 存储池模块
    - [x] 中心存储池资源管理
        - [x] 文件上传
        - [ ] 文件秒传
        - [X] 文件分片上传
    - [x] 个人存储池资源管理
        - [x] 文件列表
        - [x] 文件名称修改
        - [x] 文件夹创建
        - [ ] 文件删除
        - [x] 文件移动
- [x] 文件分享模块
    - [X] 创建分享记录
    - [X] 获取资源详情
    - [ ] 资源保存
